# visa-approval-prediction
Machine Learning classification project for visa approval prediction using Python and Scikit-learn

## Project Report

Full project analysis, business problem, code, and outputs:

➡️ [Open Full HTML Project Report](./visa_approval_prediction.html)

## Business Problem

This project aims to predict visa approval outcomes based on applicant and employment-related information.

## Objective

Develop a machine learning classification model to support decision-making and identify applicants with a higher probability of visa approval.

## Dataset

Historical visa application dataset containing applicant and employment-related variables.

## Techniques Used

* Exploratory Data Analysis (EDA)
* Feature Engineering
* Data Preprocessing
* Classification Modeling
* Hyperparameter Tuning

## Models Applied

* Decision Tree
* Random Forest
* Bagging
* Boosting

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1-score

## Technologies

Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
